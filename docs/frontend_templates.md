# Frontend templates
- HTML5 + CSS3 + vanilla JS, no SPA frameworks
- Leaflet for map (templates/map.html, static/js/map.js)
- Chart.js for KPI visualizations (dashboard, ward detail)
- Components: cards, tables, badges, pills; defined in CSS under `static/css`
- Responsive grids via `.grid` classes; print styles for reports
- Each page uses small page-specific JS to keep progressive enhancement simple
